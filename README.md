# Hi, I'm Nadira

I build AI systems because I kept running into problems that existing tools did not solve well enough.

I wanted to understand why context disappeared, why AI output was sometimes wrong or incomplete, and where a workflow actually broke. That led me to build systems that make those questions explicit: a governance engine that evaluates tool calls, an archive system that preserves project context, a reasoning agent that grounds its advice in evidence, and an application that validates and bounds AI-assisted suggestions.

I use AI as part of how I build, but not as a substitute for understanding what I ship. I test behaviour, examine assumptions, investigate failures and document where the system stops.

## Featured projects

### [Threshia Governance](https://github.com/nadira-busse/threshia-governance)

A Python governance engine that evaluates proposed AI-agent tool calls as `ALLOW`, `BLOCK` or `FLAG` before execution.

**Built with:** rule-first evaluation, ChromaDB retrieval, Mistral and OpenAI provider abstractions, audit logging, 77 tests and CI.

### [Weft](https://github.com/nadira-busse/weft)

Archive-first context infrastructure for storing AI conversations, decisions and workflow output as structured records that can be retrieved later.

**Built with:** Make, Notion, MCP, JSON Schemas, payload contracts, idempotent writes, retrieval and runtime proof.

### [Kelvior Agent Decision Gate](https://github.com/nadira-busse/kelvior-agent-decision-gate)

A Microsoft Foundry reasoning-agent MVP that produces evidence-grounded deployment-readiness assessments for other AI agents.

**Built with:** Foundry IQ, synthetic enterprise evidence, blocking rules, score caps, source traceability and an explicit human decision boundary.

### [Fenéla](https://github.com/nadira-busse/fenela)

An accountability PWA that helps someone turn one personal goal into small, workable steps and focus on one step at a time.

**Built with:** Next.js, React, TypeScript, bounded AI-assisted suggestions, safety checks, Web Push reminders, 46 tests and CI.
